# FeedSpace – Social Feed Application

A modern **social media feed application** where users can create posts, upload images, and view a scrolling feed.  
The project is built using the **MERN stack**: MongoDB, Express.js, React, and Node.js.

The UI is designed with an **Instagram-style feed layout** similar to Instagram.

---

## Features

- Create posts with text and images
- Upload images using file upload middleware
- View posts in a scrollable feed
- Responsive modern UI
- REST API for post management
- Clean card-based post layout

---

## Tech Stack

### Frontend
- React
- Vite
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- Multer (image uploads)

### Database
- MongoDB
- Mongoose
